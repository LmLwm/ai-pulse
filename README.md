# ⚡ AI Pulse

AI 新闻聚合仪表盘。自动抓取 Hacker News、Reddit、arXiv、GitHub Trending、HuggingFace、Lobsters、36氪。

每 30 分钟通过 GitHub Actions 自动更新。

## 使用

1. Fork 或创建此仓库
2. 进入 Settings → Pages → Source 选择 `main` 分支
3. 等待首次 Action 运行完成
4. 访问 `https://你的用户名.github.io/仓库名`

## 手动更新

进入 Actions → Update AI Pulse → Run workflow
